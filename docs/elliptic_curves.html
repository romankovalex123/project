Эллиптические Кривые

Математическая основа современной криптографии

Что такое эллиптическая кривая?

Эллиптическая кривая — это гладкая кривая, задаваемая уравнением вида:

y² = x³ + ax + b

В криптографии используются эллиптические кривые над конечными полями (чаще всего простыми), где уравнение принимает вид:

y² ≡ x³ + ax + b (mod p)

где p — простое число, определяющее размер поля.

Основные свойства

Симметрия

Кривая симметрична относительно оси x. Если точка (x,y) лежит на кривой, то точка (x,-y) также на ней лежит

Гладкость

Кривая не имеет самопересечений и острых углов (дискриминант не равен нулю)

Групповая структура

Точки кривой образуют абелеву группу относительно операции сложения точек

Дискретный логарифм

Задача дискретного логарифмирования на эллиптических кривых считается вычислительно сложной

Операции с точками

Сложение точек

Для двух различных точек P и Q, лежащих на кривой, их сумма R = P + Q определяется как:

  • Проводится прямая через P и Q
  • Находится третья точка пересечения с кривой
  • Отражаем эту точку относительно оси x

Удвоение точки

Для точки P, удвоение 2P = P + P определяется как:

  • Проводится касательная к кривой в точке P
  • Находится вторая точка пересечения с кривой
  • Отражаем эту точку относительно оси x

Умножение на скаляр

Умножение точки P на скаляр k (kP) выполняется как k-кратное сложение точки P самой с собой

Групповая структура

Множество точек эллиптической кривой вместе с точкой на бесконечности (нейтральный элемент) образует абелеву группу относительно операции сложения точек.

Свойства группы:

  • Замыкание: Сумма двух точек кривой также является точкой кривой
  • Ассоциативность: (P + Q) + R = P + (Q + R)
  • Коммутативность: P + Q = Q + P
  • Нейтральный элемент: Точка на бесконечности O
  • Обратный элемент: Для точки P = (x,y) обратной является -P = (x,-y)

Применение в криптографии

Протокол обмена ключами

ECDH (Elliptic Curve Diffie-Hellman) позволяет двум сторонам установить общий секретный ключ через незащищенный канал

Цифровые подписи

ECDSA (Elliptic Curve Digital Signature Algorithm) используется для создания и проверки цифровых подписей

Шифрование

ECIES (Elliptic Curve Integrated Encryption Scheme) обеспечивает конфиденциальность передаваемых данных
